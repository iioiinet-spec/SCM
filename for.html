for문이란?

같은 코드를 여러 번 반복 실행하는 문법

반복 횟수가 정해져 있을 때 가장 많이 사용한다. 배열 처리, 누적합, 별찍기, 완전탐색의 핵심이다.

어디에 쓰나?
배열 처리

인덱스 0~N-1 순서로 반복

누적합

sum += i 패턴으로 합계 계산

출력 반복

별찍기, 구구단, 문자 반복

완전탐색

모든 경우를 하나씩 검사

기본 문법 
for(초기식; 조건식; 증감식) {
    실행문;
}
동작 순서
① 초기식 실행 int i=1
↓
② 조건식 검사 i<=N
↓
참(true)
③ 실행문 실행
↓
④ 증감식 실행 i++
↑ 다시 검사
거짓(false)
⑤ 반복 종료
구성요소 설명
초기식 반복 시작값 설정 — int i=0;
조건식 반복 계속 여부 — i<10
증감식 반복 변화 — i++ i-- i+=2
핵심 패턴 5가지
① 1부터 N까지
for(int i=1;
    i<=N; i++)
② 배열 (0~N-1)
for(int i=0;
    i<N; i++)
③ 감소 반복
for(int i=10;
    i>=1; i--)
④ k씩 증가
for(int i=0;
    i<=10; i+=2)
⑤ 누적합
sum = 0;
sum += i;
배열 인덱스는 0부터 시작하므로 i<N (less than N) 패턴을 사용한다. i<=N 이면 범위 초과 오류!
