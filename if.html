if문이란?

조건에 따라 코드를 실행하는 문법

"만약 ~라면 실행한다"를 코드로 표현한다. 조건이 참(true)이면 실행, 거짓(false)이면 건너뛴다.

조건식에 쓰는 연산자

비교 연산자

> 크다   < 작다   == 같다   != 다르다   >= 이상   <= 이하

논리 연산자

&& AND (둘 다 참)   || OR (하나라도 참)   ! NOT (반대로)

주의!

== 와 = 헷갈리지 말기

== 는 비교, = 는 대입(값 넣기). if 조건식에는 반드시 == 를 사용해야 한다.

단독 if

기본 문법

if (조건) {
    실행문;
}

조건이 참이면 { } 안 실행. 거짓이면 아무것도 안 함.

조건 검사
↓
참(true)
{ } 안 실행
거짓(false)
아무것도 안 함
예제

양수면 POSITIVE 출력

#include <stdio.h>

int main() {
    int x;
    scanf("%d", &x);

    if (x > 0) {
        printf("POSITIVE");
    }

    return 0;
}
if - else

기본 문법

if (조건) {
    실행문1;
} else {
    실행문2;
}

조건이 참이면 실행문1, 거짓이면 실행문2. 둘 중 하나는 반드시 실행된다.

조건 검사
↓
참(true)
실행문1
거짓(false)
실행문2 (else)
예제

짝수면 EVEN, 아니면 ODD 출력

#include <stdio.h>

int main() {
    int x;
    scanf("%d", &x);

    if (x % 2 == 0) {
        printf("EVEN");
    } else {
        printf("ODD");
    }

    return 0;
}
if - else if - else

여러 조건 검사

if (조건1) {
    실행문1;
} else if (조건2) {
    실행문2;
} else {
    실행문3;
}

위에서부터 순서대로 검사. 처음으로 참이 되는 블록 하나만 실행. 모두 거짓이면 else 실행.

예제

점수로 학점 출력 (90→A, 80→B, 70→C, 나머지→F)

#include <stdio.h>

int main() {
    int score;
    scanf("%d", &score);

    if (score >= 90) {
        printf("A");
    } else if (score >= 80) {
        printf("B");
    } else if (score >= 70) {
        printf("C");
    } else {
        printf("F");
    }

    return 0;
}
핵심: score가 85면 score >= 80에서 멈추고 아래 조건은 검사하지 않는다.
중첩 if

if 안에 if

if (조건1) {
    if (조건2) {
        실행문;
    }
}

조건1도 만족 AND 조건2도 만족할 때 단계적으로 검사. 예: "양수이면서 짝수"

예제

양수면 POS, 양수이면서 짝수면 EVEN도 출력

#include <stdio.h>

int main() {
    int x;
    scanf("%d", &x);

    if (x > 0) {
        printf("POS\n");

        if (x % 2 == 0) {
            printf("EVEN\n");
        }
    }

    return 0;
}
&& (AND)

둘 다 참이어야 참

if (x > 0 && x % 2 == 0)
// 양수이면서 짝수
|| (OR)

하나라도 참이면 참

if (x == 0 || x == 10)
// x가 0이거나 10이면
! (NOT)

참/거짓을 반대로

if (!(x > 0))
// x가 양수가 아님 (0 또는 음수)
범위 조건

10 이상 20 이하

if (x >= 10 && x <= 20)
// AND로 묶기
범위 조건은 반드시 && (AND)로 두 조건을 연결한다.
