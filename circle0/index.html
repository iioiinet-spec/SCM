원의 방정식 시각화 실습실

(x−a)² + (y−b)² = r²

좌표평면 위에서 중심(주황 점)을 드래그하거나 가장자리 핸들로 반지름을 바꾸면(모두 정수 단위) 원의 방정식이 실시간으로 바뀝니다. 하늘색 점 P를 움직여 대입 판정(원 위/안/밖)도 확인해 보세요.

좌표평면 — 중심·핸들·점 P를 드래그하세요

a (중심 x) 2 b (중심 y) -1 r (반지름) 3
범위: 중심 a, b는 -6 ~ 6, 반지름 r은 1 ~ 6 (정수). 초록 마름모 = 좌표축과의 교점.

원의 방정식

표준형 — 중심 (a, b), 반지름 r가 그대로 보입니다
일반형 — 표준형을 전개한 x² + y² + Dx + Ey + F = 0

위치 관계 판정

학습 포인트

1. 방정식의 뿌리는 거리 공식 — 원 = 중심에서 거리가 r로 같은 점들의 모임. √((x−a)²+(y−b)²) = r의 양변을 제곱한 것이 표준형입니다.
2. 부호 함정 — 중심이 (2, −1)이면 식은 (x−2)² + (y+1)². 괄호 안 부호는 중심 좌표의 반대입니다!
3. 우변은 r² — 우변이 9라면 반지름은 9가 아니라 3. 반지름을 2배로 하면 우변은 4배가 됩니다.
4. 축과의 위치 관계 — 중심에서 x축까지 거리 |b|와 r를 비교: |b| < r 두 점 / |b| = r 접함 / |b| > r 안 만남. y축은 |a|로 같은 판정.
5. 점 대입 판정 — 점 P를 좌변에 대입한 값이 r²보다 작으면 원 안, 같으면 원 위, 크면 원 밖. 일반형에서 F = 0이면 원점을 지나는 원!
