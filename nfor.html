C언어 중첩 for문 튜토리얼

반복 안의 반복 — 별찍기부터 완전탐색까지

중첩 for문이란?

for문 안에 또 다른 for문이 들어있는 구조

"반복 안에서 또 반복"이 일어난다. 2차원 구조를 표현할 때 핵심으로 사용된다.

어디에 쓰나?
2차원 배열

행·열을 각각 반복 처리

별찍기

행=줄, 열=칸 구조

구구단 전체

단×줄 이중 반복

완전탐색

O(N²) 기본형

반복 횟수 공식

총 반복 횟수 = 바깥 반복 × 안쪽 반복

예) 바깥 i: 1~3 (3번) × 안쪽 j: 1~2 (2번) = 총 6번

기본 문법 
for(초기식1; 조건식1; 증감식1) {   ← 바깥 for (행)

    for(초기식2; 조건식2; 증감식2) { ← 안쪽 for (열)
        실행문;
    }

    printf("\n"); ← 줄바꿈은 바깥 for 안에
}
동작 흐름
① 바깥 초기식 (i=1)
↓
② 바깥 조건 검사 (i<=N)
↓ 참
③ 안쪽 초기식 (j=1)
↓
④ 안쪽 조건 검사 (j<=M)
↓ 참
⑤ 실행문 + 안쪽 증감 (j++)
↑ 반복 → 거짓이면
⑥ 줄바꿈 + 바깥 증감 (i++)
↑ 바깥 반복 → 거짓이면
⑦ 종료
핵심: 바깥 for가 1번 돌 때, 안쪽 for는 처음부터 끝까지 완전히 다 돈다.
단계별 실행 추적기

for(i=1;i<=행;i++) × for(j=1;j<=열;j++)

단계ij동작
핵심 이해 포인트
바깥 for → 행(줄) 제어
안쪽 for → 열(칸) 제어
줄바꿈 → 바깥 for 안 (안쪽 for 끝난 직후)
별 개수 → 안쪽 for 조건 변경
공백+별 → 공백 for + 별 for 두 개 사용
행×열 패턴 
for(int i=1; i<=행; i++){
    for(int j=1; j<=열; j++){
        printf("*");
    }
    printf("\n");
}
삼각형 패턴 
/* 직각삼각형: 안쪽 조건을 j<=i 로 변경 */
for(int i=1; i<=n; i++){
    for(int j=1; j<=i; j++)
        printf("*");
    printf("\n");
}
공백+별 정렬 패턴 
for(int i=1; i<=n; i++){
    for(int s=1; s<=n-i; s++) printf(" "); ← 공백
    for(int j=1; j<=i;   j++) printf("*"); ← 별
    printf("\n");
}
