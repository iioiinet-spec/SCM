정올 1071 약수와 배수

기준 수의 약수 합과 배수 합을 구하는 인터렉티브 강의자료

1. 문제 핵심

여러 개의 정수가 주어지고, 마지막에 기준 정수 m이 주어집니다. 주어진 정수들 중에서 m의 약수와 m의 배수를 찾아 각각의 합을 구합니다.

1. 어떤 수가 m을 나누어떨어지게 하면 그 수는 m의 약수입니다.

2. 어떤 수가 m으로 나누어떨어지면 그 수는 m의 배수입니다.

3. 약수의 합과 배수의 합을 따로 계산합니다.

2. 핵심 조건식

1. arr[i]가 m의 약수인지 확인

if (m % arr[i] == 0)

2. arr[i]가 m의 배수인지 확인

if (arr[i] % m == 0)
약수 판정과 배수 판정은 나누는 방향이 다릅니다.

3. 인터렉티브 계산기

숫자 목록과 기준 수 m을 입력하면 약수와 배수를 자동으로 찾아줍니다.

숫자 분류

4. 예제 손풀이

숫자 목록: 2 3 5 12 18 24
기준 수 m: 12

약수는 12를 나누어떨어지게 하는 수입니다.

12 % 2 == 0  → 2는 약수
12 % 3 == 0  → 3은 약수
12 % 5 != 0  → 5는 약수 아님
12 % 12 == 0 → 12는 약수

약수의 합은 2 + 3 + 12 = 17입니다.

배수는 12로 나누어떨어지는 수입니다.

12 % 12 == 0 → 12는 배수
24 % 12 == 0 → 24는 배수

배수의 합은 12 + 24 = 36입니다.

5. 코드 작성 방법

1. 개수 n 입력

int n;
scanf("%d", &n);

2. 배열 입력

int arr[50];

for (int i = 0; i < n; i++) {
    scanf("%d", &arr[i]);
}

3. 기준 수 m 입력

int m;
scanf("%d", &m);

4. 합 변수 준비

int divisorSum = 0;
int multipleSum = 0;

5. 약수와 배수 검사

for (int i = 0; i < n; i++) {
    if (m % arr[i] == 0) {
        divisorSum += arr[i];
    }

    if (arr[i] % m == 0) {
        multipleSum += arr[i];
    }
}

6. 자주 하는 실수

실수 1. 약수 조건을 반대로 쓰는 경우

arr[i]가 m의 약수인지 볼 때는 m을 arr[i]로 나눕니다.

예를 들어 3이 12의 약수인지 보려면 12 % 3을 해야 합니다.

실수 2. 배수 조건을 반대로 쓰는 경우

arr[i]가 m의 배수인지 볼 때는 arr[i]를 m으로 나눕니다.

예를 들어 24가 12의 배수인지 보려면 24 % 12를 해야 합니다.

실수 3. 기준 수 m을 한쪽에만 더하는 경우

m은 자기 자신의 약수이면서 배수입니다.

m % m == 0이므로 m은 약수 합에도 들어가고 배수 합에도 들어갑니다.

7. 정답 C 코드

#include <stdio.h>

int main() {
    int n;
    int arr[50];
    int m;
    int divisorSum = 0;
    int multipleSum = 0;

    scanf("%d", &n);

    for (int i = 0; i < n; i++) {
        scanf("%d", &arr[i]);
    }

    scanf("%d", &m);

    for (int i = 0; i < n; i++) {
        if (m % arr[i] == 0) {
            divisorSum += arr[i];
        }

        if (arr[i] % m == 0) {
            multipleSum += arr[i];
        }
    }

    printf("%d
", divisorSum);
    printf("%d
", multipleSum);

    return 0;
}

8. 핵심 정리

1. 약수 조건은 m % arr[i] == 0입니다.

2. 배수 조건은 arr[i] % m == 0입니다.

3. 약수 합과 배수 합은 각각 따로 더합니다.

4. 기준 수 m은 약수이면서 배수일 수 있습니다.
