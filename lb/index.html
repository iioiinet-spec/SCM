선형 탐색 vs 이진 탐색 · C / Python

배열에서 값을 찾는 두 가지 방법을 비교합니다. 선형 탐색은 앞에서부터 하나씩, 이진 탐색은 정렬된 배열에서 반씩 잘라내며 찾습니다. 언어(C/Python)를 바꿔 같은 알고리즘의 두 표현을 보고, 🏁 경주 모드에서 비교 횟수 차이를 확인하세요.

코드 · 단계 실행

선형 탐색 vs 이진 탐색 — 같은 값을 찾는 경주입니다. 원소 1개를 확인(비교)할 때마다 한 걸음씩 나아갑니다.
이진 탐색은 한 번 비교할 때마다 남은 후보가 절반으로 줄어듭니다. 값을 배열 끝쪽 수로 바꾸거나, 없는 값(예: 8)으로 바꿔서 차이를 극대화해 보세요!
원소가 1,000개라면? 선형은 최대 1,000번, 이진은 최대 10번. 10억 개라도 이진 탐색은 30번이면 끝납니다.
