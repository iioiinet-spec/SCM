C언어 변수와 입출력 튜토리얼

데이터를 담는 상자 — 변수 선언부터 scanf/printf까지

변수(Variable)란?

데이터를 저장하는 이름 붙은 메모리 공간

컴퓨터 메모리에 값을 넣어두고 이름으로 꺼내쓴다. 수학의 x, y와 같은 개념이다.

int a = 10;
10
a (int)
→
a라는 이름의 상자에
정수 10을 넣었다는 뜻
변수 선언 기본 문법 
// 선언만
int age;

// 선언 + 초기화
int age = 20;

// 여러 변수 선언
int a, b, c;
int x = 1, y = 2;
변수 이름 규칙
✓ 영문자, 숫자, 밑줄(_) 사용 가능
✓ 반드시 영문자 또는 _로 시작
✗ 숫자로 시작 불가 — 1num 오류
✗ 예약어 사용 불가 — int, for 등
자료형(Data Type) 종류
자료형저장 내용크기예시
int 정수4 byte1, -5, 100
float 실수 (소수점)4 byte3.14f, 0.5f
double 더 정밀한 실수8 byte3.141592
char 문자 1개1 byte'A', '0', '\n'
long long 큰 정수8 byte1000000000LL
C언어에는 string 타입이 없다. 문자열은 char 배열로 표현한다.
자료형 선언 예시 
int       a = 10;       // 정수
float     b = 3.14f;   // 실수 (낮은 정밀도)
double    c = 3.14159; // 실수 (높은 정밀도)
char      d = 'A';     // 문자 1개
long long e = 123456789LL; // 큰 정수
출력: printf()

화면에 값을 출력할 때 사용

printf("Hello");           // 문자열 출력
printf("%d", a);          // 정수 출력
printf("%d %d", a, b);   // 여러 값 출력
printf("a=%d\n", a);      // 줄바꿈 포함
서식 지정자 (Format Specifier)
%d
정수 (int)
%f
실수 (float/double)
%c
문자 (char)
%s
문자열 (char[])
%lld
큰 정수 (long long)
%.2f
소수점 2자리
입력: scanf()

키보드 값을 변수에 저장

scanf("%d", &a);         // 정수 1개 입력
scanf("%d %d", &a, &b); // 정수 2개 입력
scanf(" %c", &ch);        // 문자 입력 (앞 공백 중요!)
scanf("%f", &x);         // 실수 입력
⚠ scanf에는 반드시 &(주소연산자)를 붙여야 한다. 단, 배열(문자열)은 예외.
자주 하는 실수 
// 잘못된 코드
scanf("%d", a);   ← & 빠짐 → 오류!

// 올바른 코드
scanf("%d", &a);  ← & 필수

// 문자 입력 시 앞에 공백
scanf(" %c", &ch); ← 앞 공백으로 개행문자 무시
