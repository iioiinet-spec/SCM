한 줄 정의

"여러 개의 구조체를 한 번에 관리하는 가장 기본적인 방법"

구조체만 쓸 때

Student s1; Student s2; Student s3; // 30명이면 30줄...

구조체 배열을 쓰면

Student s[30]; // 30명도 한 줄! for(int i=0;i<30;i++) printf(s[i].name);

구조체

여러 정보를 하나로 묶음
이름 + 나이 = Student

배열

같은 것을 여러 개 저장
Student[30]

구조체 배열

같은 구조를 여러 개
학생 30명 관리

실제로 쓰이는 곳

  • 학생 관리 시스템 (이름 + 점수 + 학번)
  • 상품 목록 (이름 + 가격 + 재고)
  • 게임 캐릭터 목록 (이름 + 체력 + 레벨)
  • 지도 좌표 데이터 (x + y + 레이블)

선언 형식

typedef struct { char name[20]; // 멤버 1 int age; // 멤버 2 } Student; // 구조체 이름 Student s[3]; // 구조체 배열 선언

접근 방법 — 배열[인덱스].멤버

s[0].name → 0번째 학생의 이름
s[1].age → 1번째 학생의 나이
s[i].name → i번째 학생의 이름 (반복문)

문자열 저장 — strcpy 사용

strcpy(s[0].name, "Kim"); // s[0].name = "Kim"; 은 틀림!

숫자 저장 — 직접 대입

s[0].age = 14; // 숫자는 바로 대입 가능

메모리 안에서의 배치

구조체 배열은 메모리에서 각 구조체가 연속으로 배치됩니다. Student 하나는 name(20바이트) + age(4바이트) = 24바이트입니다.

배열 인덱스 계산

s[i] 시작 주소
= s[0] + i × sizeof(Student)

멤버 접근

s[i].name → 구조체 시작
s[i].age → +20바이트 위치

크기 비교

char name[20]
20 bytes
int age
4 bytes
Student 1개
≈ 24 bytes
Student[3]
≈ 72 bytes

학생 3명 이름·나이 저장 및 출력

Student s[3]에 Kim(14), Lee(15), Park(16)을 저장하고 반복문으로 출력합니다.

typedef struct { char name[20]; int age; } Student; int main() { Student s[3]; strcpy(s[0].name, "Kim"); s[0].age = 14; strcpy(s[1].name, "Lee"); s[1].age = 15; strcpy(s[2].name, "Park"); s[2].age = 16; for(int i=0; i<3; i++) printf("이름: %s, 나이: %d\n", s[i].name, s[i].age); return 0; }

실행 결과

이름: Kim, 나이: 14 이름: Lee, 나이: 15 이름: Park, 나이: 16

점 3개의 좌표 저장 및 출력

Point p[3]에 (1,2), (3,4), (5,6) 저장 후 반복문으로 출력합니다.

typedef struct { int x; int y; } Point; int main() { Point p[3]; p[0].x=1; p[0].y=2; p[1].x=3; p[1].y=4; p[2].x=5; p[2].y=6; for(int i=0; i<3; i++) printf("(%d, %d)\n", p[i].x, p[i].y); return 0; }

실행 결과

(1, 2) (3, 4) (5, 6)

상품 2개 입력받아 출력

scanf로 이름과 가격을 입력받고 반복문으로 출력합니다.

typedef struct { char name[20]; int price; } Product; int main() { Product p[2]; for(int i=0; i<2; i++) { scanf("%s", p[i].name); scanf("%d", &p[i].price); } for(int i=0; i<2; i++) printf("이름: %s, 가격: %d원\n", p[i].name, p[i].price); return 0; }

입력 예시 → 출력

입력
Apple 1500 Banana 800
출력
이름: Apple, 가격: 1500원 이름: Banana, 가격: 800원

핵심 정리

  • 구조체 배열은 구조체를 여러 개 저장하는 배열
  • 선언: 구조체이름 배열이름[개수]
  • 접근: 배열[i].멤버
  • 반복문과 함께 쓰면 코드가 매우 간결해짐
  • 문자열은 strcpy, 숫자는 직접 대입 또는 scanf
