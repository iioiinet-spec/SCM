구조체란?
01구조체(struct)의 정의
서로 다른 자료형의 변수를 하나로 묶는 사용자 정의 자료형입니다.
배열은 같은 자료형만 저장하지만, 구조체는 int·char·float 등을 함께 묶을 수 있습니다.
C — 구조체 기본 형식
struct Student {
    char  name[20];  // 이름 (문자열)
    int   age;        // 나이 (정수)
    float height;     // 키  (실수)
};                   // ← 세미콜론 필수!
02구조체 변수 선언 · 멤버 접근
구조체 자료형으로 변수를 선언하고, . 연산자로 멤버에 접근합니다.
C — 선언 · 접근 · 출력
struct Student s1;      // 구조체 변수 선언

strcpy(s1.name, "Kim");  // 문자열 → strcpy
s1.age    = 14;           // 정수   → 직접 대입
s1.height = 160.5;        // 실수   → 직접 대입

printf("%s\n", s1.name);
printf("%d\n", s1.age);
printf("%.1f\n", s1.height);
03구조체 배열 — 여러 명 관리
구조체와 배열을 결합하면 여러 개의 데이터를 체계적으로 관리할 수 있습니다.
C — 구조체 배열
struct Student s[3];   // 학생 3명

strcpy(s[0].name, "Kim");
s[0].age = 14;

strcpy(s[1].name, "Lee");
s[1].age = 15;

printf("%s %d\n", s[0].name, s[0].age);
printf("%s %d\n", s[1].name, s[1].age);
04완성 예제 — 학생 정보 저장 · 출력
C — 전체 코드
#include <stdio.h>
#include <string.h>

struct Student {
    char name[20];
    int  age;
    int  score;
};

int main() {
    struct Student s;

    strcpy(s.name, "Jisu");
    s.age   = 14;
    s.score = 95;

    printf("이름: %s\n", s.name);
    printf("나이: %d\n", s.age);
    printf("점수: %d\n", s.score);

    return 0;
}
실행 결과:
이름: Jisu
나이: 14
점수: 95
05배열 vs 구조체 차이
배열(array)
int arr[5];
같은 자료형 여러 개
구조체(struct)
struct S {
  char n[20];
  int  a;
};
다른 자료형 한 묶음
구조체 빌더 — 멤버를 입력하고 구조체 메모리 구조와 출력 확인
값을 입력하고 실행 버튼을 누르세요
scanf 입력 방식 체험 — 문자열 vs 숫자 멤버 차이
입력 방식 비교
// 문자열 배열 → & 없음
scanf("%s", p.name);

// 정수/실수 → & 필요
scanf("%d", &p.price);
scanf("%f", &p.height);
배열 이름(name)은 이미 주소값이므로 &를 붙이지 않습니다.
int, float 등 일반 변수는 반드시 &변수명으로 주소를 전달해야 합니다.
자주 하는 실수 4가지
실수 1구조체 정의 끝 세미콜론 빠뜨리기
틀린 예: struct Student { ... } ← ; 없음
맞는 예: struct Student { ... }; ← 반드시 ; 필요
구조체 정의는 선언문이므로 중괄호 뒤에 반드시 세미콜론이 와야 합니다.
실수 2문자열 멤버에 = 대입 사용
틀린 예: s1.name = "Kim"; ← 문자열은 = 대입 불가
맞는 예: strcpy(s1.name, "Kim"); ← strcpy 사용
char 배열은 포인터가 아니라 배열이므로 = 로 직접 대입할 수 없습니다. strcpy를 사용해야 합니다.
실수 3. 연산자 빠뜨리기
틀린 예: s1 age = 14; ← . 없음
맞는 예: s1.age = 14; ← . 으로 멤버 접근
구조체 변수와 멤버 이름 사이에는 반드시 . 이 있어야 합니다.
실수 4scanf에서 & 잘못 사용
틀린 예: scanf("%s", &p.name); ← 문자열에 & 붙이면 경고
틀린 예: scanf("%d", p.price); ← 정수에 & 없으면 오류
맞는 예: scanf("%s", p.name);
맞는 예: scanf("%d", &p.price);
정리실수 방지 체크리스트
구조체 닫는 중괄호 뒤
→ ; 필수
문자열 대입
→ strcpy 사용
멤버 접근
→ 변수.멤버
scanf 숫자
→ &변수 필수
