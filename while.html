C언어 while문 튜토리얼

조건 반복의 핵심 — 입력 처리부터 자릿수 분해까지

while문이란?

조건이 참인 동안 계속 반복하는 반복문

for문은 횟수 반복에 강하고, while문은 조건이 유지되는 동안 반복에 강하다.

대표 사용처
센티넬 입력

특정 값이 나오면 종료

값 감소 반복

0이 될 때까지 반복

자릿수 분해

10으로 나누기 반복

EOF 입력

입력 끝날 때까지 반복

무한루프 주의!

조건을 바꾸는 코드가 반드시 있어야 한다

while (x > 0) {
    x--; ← 이게 없으면 영원히 반복
}
기본 문법 
while (조건) {
    실행문;
    // 조건을 바꾸는 코드
}
동작 순서
① 조건 검사
참(true) ↓
② 실행문 실행
↓
③ 조건 변화 (i++)
↑ 다시 조건 검사
거짓(false) →
④ 즉시 종료
조건을 먼저 검사하므로, 처음부터 거짓이면 한 번도 실행되지 않는다.
do-while 비교 
/* while: 조건 먼저 → 0번 가능 */
while (조건) { 실행문; }

/* do-while: 일단 실행 → 최소 1번 */
do { 실행문; } while (조건);
for vs while 비교
구분for문while문
강점횟수가 정해진 반복조건이 유지되는 동안 반복
초기식for 헤더에 포함while 앞에 별도 선언
증감식for 헤더에 포함실행문 안에 직접 작성
0회 실행가능가능
대표 패턴1~N 출력, 배열 순회센티넬 입력, 자릿수 분해, EOF
같은 결과, 다른 문법
for문
for(int i=1; i<=5; i++)
    printf("%d ", i);
while문
int i = 1;
while (i <= 5) {
    printf("%d ", i);
    i++;
}
while 핵심 패턴 4가지
① 횟수 반복 (for 대체)
i = 1;
while(i<=N) {
    ...
    i++;
}
② 센티넬 입력
while(1) {
    scanf(...);
    if(종료값)
        break;
    ...
}
③ 자릿수 처리
while(n > 0) {
    n%10;   // 1의 자리
    n/=10;  // 자리 제거
}
④ EOF 입력
while(
  scanf("%d",&x)
  ==1) {
    ...
}
패턴 ②에서 while(1)은 조건이 항상 참이므로 무한루프. 반드시 break로 탈출해야 한다.
