① 영화 설정 생성 프롬프트

ChatGPT 프롬프트
나는 초등학생이야. 내가 만들 영화 포스터를 위해 아래 정보를 만들어줘: 1. 영화 제목 (한글, 멋지고 기억에 남는 것) 2. 한 줄 카피 (포스터 하단에 쓸 짧은 문장) 3. 주인공 이름과 특징 1가지 4. 포스터 배경색 추천 (어두운 톤 2가지, 밝은 톤 2가지) 5. 포스터 분위기 설명 (예: 미래도시, 마법의 숲, 우주 등) SF 어드벤처 장르로 만들어줘.
💡 팁: 장르를 바꿔보세요! "공포", "로맨스", "판타지" 등으로 변경하면 전혀 다른 포스터가 나와요.
Gemini 프롬프트
안녕 Gemini! 나는 중학생이고 영화 포스터를 HTML로 만들 거야. 다음 내용을 JSON 형식으로 만들어줘: { "title": "영화 제목 (강렬하고 짧게)", "tagline": "포스터 카피 (20자 이내)", "hero": "주인공 이름과 능력", "palette": { "bg_dark": "헥스 코드", "bg_light": "헥스 코드", "accent": "포인트 컬러 헥스 코드", "text": "#ffffff" }, "mood": "분위기 키워드 3개" } 장르: 사이버펑크 액션
💡 팁: JSON 형식으로 받으면 HTML 코드에 바로 변수로 넣기 쉬워요!

② HTML 포스터 코드 생성 프롬프트

ChatGPT 프롬프트
위에서 만든 설정으로 HTML 파일 하나를 만들어줘. 조건: - 외부 라이브러리 없이 순수 HTML + CSS + JS만 사용 - 포스터 크기: 세로형 (width: 400px, height: 600px) - 배경: CSS linear-gradient로 영화 분위기 표현 - 제목 텍스트: 크고 두꺼운 폰트 (Google Fonts 사용 가능) - 포스터 하단에 카피 문구 배치 - 마우스를 올리면 글자가 빛나는 효과 추가 - 코드 전체를 하나의 HTML 파일로 작성해줘
💡 팁: "마우스 올리면 글자가 빛나는 효과" 같은 인터랙션 조건을 추가할수록 더 멋진 결과가 나와요!
Gemini 프롬프트
앞에서 만든 영화 설정을 이용해서 완성된 HTML 영화 포스터 페이지를 만들어줘. 요구사항: 1. 파일 하나 (index.html)로 완성 2. 세로형 포스터 (400×600px) 중앙 배치 3. CSS animation으로 제목 타이핑 효과 넣기 4. 포스터 아래에 "예매하기" 버튼 (클릭 시 alert) 5. 반응형 (모바일에서도 보기 좋게) 6. 다크 테마 배경 영화 제목: [앞에서 만든 제목] 카피: [앞에서 만든 카피] 배색: [앞에서 받은 팔레트]
💡 팁: 앞 단계에서 받은 제목·카피·색상을 [ ] 안에 직접 채워 넣어요!

① 파티클 아트 기본 코드 생성

ChatGPT 프롬프트
HTML Canvas를 사용해서 인터랙티브 미디어아트를 만들어줘. 기능: - 화면에 100개의 작은 원(파티클)이 떠다님 - 마우스를 움직이면 파티클들이 마우스를 피해서 도망감 - 파티클이 화면 가장자리에서 튕겨나옴 - 파티클 색상은 무지개 그라디언트 (hsl 사용) - 클릭하면 파티클이 마우스 위치로 폭발처럼 모임 - 배경: 검은색 외부 라이브러리 없이 순수 JS Canvas로 만들어줘. 단일 HTML 파일로 완성해줘.
💡 팁: 파티클 수(100)를 바꾸거나 "도망감" 대신 "따라옴"으로 바꿔보세요!
Gemini 프롬프트
안녕! 나는 미디어아트 작품을 만들고 싶어. HTML + Canvas로 아름다운 인터랙티브 작품을 만들어줘. 테마: 별빛 은하수 요소: 1. 배경에 별들이 천천히 반짝임 (투명도 애니메이션) 2. 마우스를 움직이면 별똥별 궤적이 생김 3. 마우스 클릭 시 별이 폭발하는 파티클 이펙트 4. 화면 중앙에 "MY UNIVERSE"라는 텍스트가 별빛과 함께 빛남 5. 스페이스바를 누르면 색상 테마 변경 (파랑→보라→초록) 코드는 주석과 함께 깔끔하게 작성해줘.
💡 팁: 테마를 "바다 속", "불꽃", "봄 꽃잎" 등으로 바꿔보세요!

② 타이포그래피 아트 프롬프트

ChatGPT 프롬프트
Canvas로 타이포그래피 미디어아트를 만들어줘. 효과: - 화면 중앙에 큰 텍스트 "VIBE"를 표시 - 텍스트가 파티클(점)들로 이루어져 있음 - 처음에는 파티클들이 흩어져 있다가 모여서 글자 모양 완성 - 마우스를 올리면 파티클들이 다시 흩어짐 - 파티클 색상: 청록색 계열 - 폰트: 굵은 산세리프 단계별 설명 주석 포함해서 작성해줘.
💡 팁: "VIBE" 대신 자기 이름, 학교 이름, 좋아하는 단어로 바꿔보세요!
Gemini 프롬프트
HTML Canvas로 인터랙티브 타이포그래피 아트를 만들어줘. 작품 개념: "내 이름이 살아움직인다" 구체적 효과: 1. 이름 "김민준"을 픽셀/파티클로 표현 2. 배경음악 없이 시각적으로만 '리듬감' 표현 (파티클이 파도처럼 출렁임) 3. 마우스 위치에 따라 글자가 물결처럼 변형됨 4. 더블클릭 시 폭발 → 재조합 애니메이션 5. 색상 그라디언트: 코랄 → 황금색 완전한 single HTML 파일로 만들어줘.
💡 팁: "김민준" 부분에 자신의 이름을 넣어요!

① 피하기 게임 만들기

ChatGPT 프롬프트
HTML Canvas로 간단한 피하기 게임을 만들어줘. 게임 규칙: - 플레이어: 파란 사각형, 마우스로 이동 - 적: 빨간 원들이 위에서 아래로 떨어짐 - 닿으면 게임 오버 - 살아남은 시간이 점수 (초 단위) - 시간이 지날수록 적이 빠르고 많아짐 - 스페이스바로 시작/재시작 UI 포함: - 점수판 (상단) - 게임오버 화면 (최고점수 표시) - 귀여운 픽셀 스타일 디자인 전체 코드를 HTML 파일 하나로 작성해줘.
💡 팁: "파란 사각형"을 "캐릭터 이모지 🐱"로 바꿔달라고 추가 요청해보세요!
Gemini 프롬프트
안녕! 재미있는 게임을 HTML로 만들고 싶어. 게임 이름: 별 먹기 대작전 게임 방식: 1. 캐릭터(🚀)를 방향키로 움직임 2. 화면에 나타나는 ⭐를 먹으면 +10점 3. 💣을 맞으면 목숨 1개 감소 (총 3개) 4. 별을 10개 먹을 때마다 레벨업 (속도 증가) 5. 목숨이 0이 되면 게임오버 필요한 것: - 점수판, 목숨 표시 - 레벨업 이펙트 (화면 잠깐 번쩍) - 게임오버 시 순위 저장 (localStorage) - 귀여운 UI 디자인 완전한 HTML 파일로 만들어줘!
💡 팁: 이모지를 바꿔서 완전히 다른 테마의 게임을 만들 수 있어요!

② 퀴즈 게임 만들기

ChatGPT 프롬프트
재미있는 퀴즈 게임 HTML을 만들어줘. 퀴즈 주제: 한국 역사 (초등학교 수준) 문제 수: 10문제 형식: 4지선다 기능: - 제한 시간 15초 (타이머 바 표시) - 정답 시 초록 효과, 오답 시 빨간 효과 - 점수 누적 표시 - 마지막에 총점 & 등급 (100점=S, 80점=A, 60점=B, 그 이하=C) - 틀린 문제 복습 기능 디자인: 밝고 귀여운 게임 스타일 (문제는 직접 넣어줘, 한국사 기초 내용으로)
💡 팁: 주제를 "수학 구구단", "영어 단어", "과학 상식"으로 바꾸면 공부 게임이 돼요!
Gemini 프롬프트
나만의 퀴즈 게임을 만들어줘! 주제: 내가 좋아하는 아이돌 (예시: BTS) 문제 유형 혼합: - 객관식 4지선다 (7문제) - OX 퀴즈 (3문제) 게임 특징: 1. 오프닝 화면 (게임 제목 + 시작 버튼) 2. 콤보 시스템 (연속 정답 시 보너스 점수) 3. 정답 후 짧은 해설 표시 4. 배경음: Web Audio API로 간단한 비프음 5. 결과 화면에 SNS 공유 텍스트 생성 HTML 파일 하나로 완성해줘. 퀴즈 내용도 BTS 관련으로 직접 만들어줘.

① 막대 비주얼라이저

ChatGPT 프롬프트
HTML + Web Audio API로 뮤직 비주얼라이저를 만들어줘. 기능: - mp3 파일을 업로드하면 재생되면서 시각화 시작 - 주파수 분석(AnalyserNode)으로 막대 그래프 표시 - 막대 수: 64개 - 막대 색상: 낮은 주파수=파랑, 높은 주파수=빨강 (hsl로 그라디언트) - 배경: 검은색 - 막대가 비트에 맞춰 위아래로 통통 튀는 느낌 - 재생/일시정지 버튼 포함 깔끔한 UI 디자인 포함해서 단일 HTML 파일로 만들어줘.
💡 팁: 막대 수를 128개로 늘리면 더 세밀한 시각화가 돼요!
Gemini 프롬프트
Web Audio API를 이용한 뮤직 비주얼라이저 HTML을 만들어줘. 시각화 스타일: 원형 비주얼라이저 - 화면 중앙에 원이 있고 주변에 막대들이 방사형으로 뻗어나옴 - 비트가 강할수록 막대가 길어지고 중앙 원이 커짐 - 컬러 테마: 네온 사이버펑크 (핑크, 청록, 보라) - 마이크 입력도 지원 (버튼으로 전환) - 파티클 이펙트: 강한 비트에서 반짝임 추가 UI: - 상단에 노래 제목 표시 (파일명에서 추출) - 볼륨 조절 슬라이더 - 풀스크린 버튼 완전한 HTML 파일로 작성해줘.
💡 팁: "마이크 입력 지원" 기능으로 목소리나 악기 소리도 시각화할 수 있어요!

3
색상과 모양 바꿔보기
막대 색상의 hsl 값만 바꿔도 완전히 다른 작품이 돼요. 숫자를 바꾸며 어떻게 변하는지 실험해요.
4
나만의 테마로 개인화
