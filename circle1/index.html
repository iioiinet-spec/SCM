원과 직선의 위치 관계 실습실

판별식 D vs 거리 d 비교

원(중심·반지름)과 직선 y = mx + n(기울기 핸들·직선 몸통 드래그, 모두 정수)을 움직이면, ① 대입 → 판별식과 ② 중심-직선 거리 d vs r 두 방법이 나란히 계산되고 같은 결론에 도달하는 것을 확인할 수 있습니다.

좌표평면 — 원과 직선을 드래그하세요

a 0 b 0 r 5 m 1 n 1
직선 몸통 드래그 = n(y절편) 이동 · 보라 네모 핸들 드래그 = 기울기 m 변경 · 분홍 점선 = 중심에서 직선까지의 수선(거리 d)

판정 결과

방법 ① — 대입해서 판별식

방법 ② — 중심과 직선 사이 거리

학습 포인트

1. 방법 ① 판별식 — 직선을 원에 대입하면 x에 대한 이차방정식. 실근의 개수 = 교점의 개수이므로 D > 0 두 점(할선), D = 0 접선, D < 0 만나지 않음.
2. 방법 ② 거리 비교 — 중심 (a, b)에서 직선 mx − y + n = 0까지 d = |ma − b + n| / √(m²+1). d < r 두 점, d = r 접선, d > r 만나지 않음.
3. 언제 무엇을? — 판정만 필요하면 거리 비교가 계산이 빠르고, 교점의 좌표까지 구해야 하면 대입(판별식) 방법을 씁니다. 두 방법의 결론은 항상 일치!
4. 접선 조건 — 원점 중심 원 x²+y²=r²과 y=mx+n이 접하려면 n² = r²(m²+1).
5. 주의 — 이 실습기의 직선은 y = mx + n 꼴이라 수직선(x = k)은 다루지 않습니다. 수직선은 |a − k|와 r를 비교하면 됩니다.
