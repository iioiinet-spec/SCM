정올 2814 이진수

2진수 문자열을 10진수로 바꾸는 인터렉티브 강의자료

1. 문제 핵심

2진수로 입력된 수를 10진수로 바꾸어 출력하는 문제입니다.

1. 입력은 0과 1로 이루어진 문자열입니다.

2. 문자열을 왼쪽부터 한 글자씩 읽습니다.

3. 현재까지 만든 10진수 값을 2배 하고 새 숫자를 더합니다.

decimal = decimal * 2 + 현재숫자

2. 인터렉티브 변환기

2진수를 입력하면 10진수로 바뀌는 과정을 보여줍니다.

입력된 비트

3. 1010 손풀이

1010₂ = 1×8 + 0×4 + 1×2 + 0×1
      = 8 + 0 + 2 + 0
      = 10

코드에서는 오른쪽 자리값을 직접 계산하지 않고 왼쪽부터 읽으며 누적합니다.

decimal = decimal * 2 + 현재숫자

4. 코드 작성 방법

1. 2진수를 문자열로 입력받기

char binary[40];
scanf("%s", binary);

2. 결과 변수 준비

int decimal = 0;

3. 문자열 끝까지 반복

for (int i = 0; binary[i] != '\0'; i++) {
    decimal = decimal * 2 + (binary[i] - '0');
}

4. 문자를 숫자로 바꾸기

binary[i] - '0'

1. '0' - '0' = 0

2. '1' - '0' = 1

5. 왜 decimal * 2를 하는가?

2진수에서 한 칸 왼쪽으로 이동한다는 것은 값이 2배가 된다는 뜻입니다.

새로운 비트를 읽기 전에 기존 값에 2를 곱하고, 새 비트 값을 더합니다.

6. 자주 하는 실수

실수 1. 입력을 int로 받는 경우

2진수는 문자열로 입력받는 것이 안전합니다.

1010을 int로 받으면 각 자리 문자 처리가 어렵습니다. 문자열로 받아야 '1', '0'을 하나씩 확인할 수 있습니다.

실수 2. 문자 '1'을 숫자 1로 착각하는 경우

입력된 한 글자는 문자입니다.

문자 '1'은 숫자 1과 다릅니다. binary[i] - '0'으로 숫자로 바꿔야 합니다.

실수 3. 거듭제곱을 복잡하게 계산하는 경우

왼쪽부터 누적하면 간단합니다.

decimal = decimal * 2 + 현재숫자 공식을 쓰면 2의 거듭제곱을 따로 계산하지 않아도 됩니다.

7. 정답 C 코드

#include <stdio.h>

int main() {
    char binary[40];
    int decimal = 0;

    scanf("%s", binary);

    for (int i = 0; binary[i] != '\0'; i++) {
        decimal = decimal * 2 + (binary[i] - '0');
    }

    printf("%d\n", decimal);

    return 0;
}

8. 핵심 정리

1. 입력은 2진수 문자열입니다.

2. 출력은 10진수 정수입니다.

3. 문자는 binary[i] - '0'으로 숫자로 바꿉니다.

4. 핵심 공식은 decimal = decimal * 2 + 현재숫자입니다.
