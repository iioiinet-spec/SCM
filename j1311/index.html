정올 카드게임

9가지 점수 규칙을 직접 확인하는 인터렉티브 강의자료

1. 문제 핵심

카드 5장을 입력받아 9가지 규칙 중 가장 높은 점수가 되는 규칙을 적용합니다.

1. 색깔은 R, B, Y, G 네 가지입니다.

2. 숫자는 1부터 9까지입니다.

3. 두 가지 이상의 규칙이 가능하면 더 높은 점수 규칙을 적용합니다.

4. 그래서 900점대 규칙부터 100점대 규칙까지 순서대로 검사하면 됩니다.

2. 인터렉티브 카드 입력

카드 5장을 입력한 뒤 점수 계산 버튼을 누르세요.

1번
2번
3번
4번
5번

3. 규칙 판정표

현재 카드가 어떤 규칙에 해당하는지 자동으로 표시됩니다.

4. 코드 작성 방법

1. 숫자 개수 세기

int count[10] = {0};

for (int i = 0; i < 5; i++) {
    count[number[i]]++;
}

2. 색깔이 모두 같은지 확인

int sameColor = 1;

for (int i = 1; i < 5; i++) {
    if (color[i] != color[0]) {
        sameColor = 0;
    }
}

3. 숫자가 연속인지 확인

for (int start = 1; start <= 5; start++) {
    int ok = 1;

    for (int j = 0; j < 5; j++) {
        if (count[start + j] != 1) {
            ok = 0;
        }
    }

    if (ok == 1) {
        straight = 1;
        high = start + 4;
    }
}

4. 같은 숫자 개수 찾기

for (int i = 1; i <= 9; i++) {
    if (count[i] == 4) four = i;
    else if (count[i] == 3) three = i;
    else if (count[i] == 2) {
        if (pair1 == 0) pair1 = i;
        else pair2 = i;
    }
}

5. 자주 하는 실수

실수 1. 낮은 규칙부터 검사하는 경우

예를 들어 모두 같은 색이면서 연속이면 600점 규칙도 가능하지만 900점 규칙이 정답입니다.

항상 900점대부터 100점대 순서로 검사해야 가장 높은 점수가 나옵니다.

실수 2. 연속 판정에서 중복 숫자를 허용하는 경우

연속 숫자는 5개의 서로 다른 숫자가 이어져야 합니다.

1 2 3 4 4는 숫자가 이어져 보이지만 5가 없고 4가 중복이므로 연속이 아닙니다.

실수 3. 두 쌍 점수에서 큰 쌍과 작은 쌍을 반대로 쓰는 경우

두 쌍은 큰 숫자 × 10 + 작은 숫자를 사용합니다.

예를 들어 5 한 쌍, 4 한 쌍이면 300 + 5×10 + 4 = 354입니다.

6. 정답 C 코드

#include <stdio.h>

int main() {
    char color[5];
    int number[5];
    int count[10] = {0};

    for (int i = 0; i < 5; i++) {
        scanf(" %c %d", &color[i], &number[i]);
        count[number[i]]++;
    }

    int sameColor = 1;

    for (int i = 1; i < 5; i++) {
        if (color[i] != color[0]) {
            sameColor = 0;
        }
    }

    int maxNum = 0;

    for (int i = 1; i <= 9; i++) {
        if (count[i] > 0) {
            maxNum = i;
        }
    }

    int straight = 0;
    int high = 0;

    for (int start = 1; start <= 5; start++) {
        int ok = 1;

        for (int j = 0; j < 5; j++) {
            if (count[start + j] != 1) {
                ok = 0;
            }
        }

        if (ok == 1) {
            straight = 1;
            high = start + 4;
        }
    }

    int four = 0;
    int three = 0;
    int pair1 = 0;
    int pair2 = 0;

    for (int i = 1; i <= 9; i++) {
        if (count[i] == 4) {
            four = i;
        } else if (count[i] == 3) {
            three = i;
        } else if (count[i] == 2) {
            if (pair1 == 0) {
                pair1 = i;
            } else {
                pair2 = i;
            }
        }
    }

    int score;

    if (sameColor && straight) {
        score = 900 + high;
    } else if (four > 0) {
        score = 800 + four;
    } else if (three > 0 && pair1 > 0) {
        score = 700 + three * 10 + pair1;
    } else if (sameColor) {
        score = 600 + maxNum;
    } else if (straight) {
        score = 500 + high;
    } else if (three > 0) {
        score = 400 + three;
    } else if (pair1 > 0 && pair2 > 0) {
        int big = pair1 > pair2 ? pair1 : pair2;
        int small = pair1 < pair2 ? pair1 : pair2;

        score = 300 + big * 10 + small;
    } else if (pair1 > 0) {
        score = 200 + pair1;
    } else {
        score = 100 + maxNum;
    }

    printf("%d\n", score);

    return 0;
}

7. 핵심 정리

1. 색깔이 모두 같은지 확인합니다.

2. 숫자가 연속인지 확인합니다.

3. 숫자별 개수를 셉니다.

4. 4장, 3장, 2장 같은 숫자를 찾습니다.

5. 높은 점수 규칙부터 검사합니다.
